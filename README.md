# ✦ Portfólio — Pietra Silveira (v2)

Design editorial escuro com detalhes dourado-rosê.

## 📁 Estrutura

```
portfolio-pietra/
├── index.html        ← Arquivo principal (4 páginas)
├── css/
│   └── style.css     ← Estilos (dark editorial + ouro)
├── js/
│   └── main.js       ← Partículas, cursor, animações
└── README.md
```

## ✨ Diferenciais deste portfólio

- **Navegação lateral vertical** — incomum e marcante
- **Canvas com partículas animadas** — feitas com JavaScript puro
- **Cursor personalizado** com efeito de lag suave
- **Preview ao hover** nos projetos — o emoji aparece junto ao cursor
- **Ticker animado** no rodapé com suas tecnologias
- **Tipografia dramática** (Playfair Display + Barlow)
- **Tema escuro editorial** com dourado e lilás

## 🚀 Publicar no Render

1. Suba a pasta no GitHub
2. Acesse render.com → **New → Static Site**
3. Conecte o repositório
4. **Publish Directory:** `.`
5. **Build Command:** (vazio)
6. Deploy!
